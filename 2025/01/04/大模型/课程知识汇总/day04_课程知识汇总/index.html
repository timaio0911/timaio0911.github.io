Day04 课程知识汇总：大模型应用开发（LlamaIndex）

课程概述

本课程是"第五阶段 第4天"的内容，主要讲解大模型应用开发，包括大模型应用的概念与类型、LlamaIndex框架的使用、RAG系统搭建、Agent智能体开发等。

目录

  1. 大模型应用概述
  2. LlamaIndex框架
  3. RAG检索增强生成
  4. Agent智能体
  5. 代码实现解析

一、大模型应用概述

1.1 什么是大模型应用？

大模型应用是以LLM为核心，通过结合外部数据、记忆机制和工具调用，来解决特定业务场景问题的软件系统。

1.2 大模型应用类型

类型 说明 典型应用
Prompt Engineering 通过设计提示词引导模型 翻译助手、文案生成器
对话系统 增加记忆功能，支持多轮对话 ChatGPT、智能客服
RAG 检索增强生成，外挂知识库 企业知识库问答
Agent 具备推理和工具调用能力 智能助手、自动化工具

1.3 构建大模型应用的挑战

挑战 描述
数据连接 如何将私有数据与LLM连接
上下文管理 如何处理长对话、保持上下文一致性
工具调用 如何让LLM调用外部API、数据库
多步骤推理 如何实现复杂的决策链和任务分解
幻觉问题 如何防止模型生成错误信息

二、LlamaIndex框架

2.1 什么是LlamaIndex？

LlamaIndex是一个专为大型语言模型应用设计的数据框架，核心使命是将私有数据与LLM连接起来。

相关链接：

  • 官网：https://www.llamaindex.ai/
  • 文档：https://docs.llamaindex.org/
  • GitHub：https://github.com/run-llama/llama_index

2.2 核心组件

组件 说明
Data Connectors 连接各种数据源，加载文档数据
Index 对数据建立索引结构，便于高效检索
Retriever 根据查询从索引中检索相关文档
Query Engine 处理查询请求，返回响应结果
Agents 具备推理和工具调用能力的智能代理
Workflows 编排多步骤任务的可编程流程

2.3 安装LlamaIndex

# 基础安装
pip install llama-index

# 按需安装特定模型支持
pip install llama-index-llms-ollama      # Ollama支持
pip install llama-index-llms-openai      # OpenAI支持
pip install llama-index-embeddings-ollama  # Ollama嵌入模型

2.4 加载模型

加载Ollama本地模型：

from llama_index.llms.ollama import Ollama

llm = Ollama(
    model="qwen3.5:0.8b",
    base_url="http://localhost:11434",
    request_timeout=120,
    context_window=1024
)

加载通义千问API：

from llama_index.llms.dashscope import DashScope

llm = DashScope(
    model_name="qwen-turbo",
    api_key="your-api-key"
)

三、RAG检索增强生成

3.1 什么是RAG？

**RAG（Retrieval-Augmented Generation，检索增强生成）**是在回答问题前，先从知识库中检索相关信息，再结合检索结果生成答案的技术。

解决的问题：

  • 大模型不了解企业内部文档
  • 大模型可能产生"幻觉"（编造信息）

3.2 RAG流程

用户问题
    ↓
[检索器] → 从知识库中检索相关文档
    ↓
[上下文拼接] → 将检索结果与问题拼接
    ↓
[大模型] → 基于检索结果生成答案
    ↓
输出答案

3.3 RAG核心步骤

步骤 说明 工具
文档加载 读取各类格式文档 PyMuPDFReader, SimpleDirectoryReader
文本分块 将长文档切分为合适大小 chunk_size, chunk_overlap
文本向量化 将文本转换为向量表示 OllamaEmbedding, OpenAIEmbedding
索引构建 建立向量索引 VectorStoreIndex
查询引擎 处理用户查询 QueryEngine

3.4 RAG代码示例

from llama_index.embeddings.ollama import OllamaEmbedding
from llama_index.readers.file import PyMuPDFReader
from llama_index.vector_stores.chroma import ChromaVectorStore
from llama_index.core import Settings, StorageContext, VectorStoreIndex
import chromadb

# 设置嵌入模型
Settings.embed_model = OllamaEmbedding(
    model_name="bge-m3:latest",
    base_url="http://localhost:11434"
)

# 加载文档
documents = PyMuPDFReader().load_data(file_path="data/麦当劳工作手册.pdf")

# 设置分块参数
Settings.chunk_size = 600
Settings.chunk_overlap = 100

# 创建向量库
db_client = chromadb.PersistentClient(path="vector_story/embedding_db")
db_collection = db_client.get_or_create_collection('test001')
vector_store = ChromaVectorStore(chroma_collection=db_collection)
storage_context = StorageContext.from_defaults(vector_store=vector_store)

# 将文档存入向量库
index = VectorStoreIndex.from_documents(documents, storage_context=storage_context)
print("文章入库成功")

3.5 查询文档

# 创建查询引擎
query_engine = index.as_query_engine()

# 查询
response = query_engine.query("麦当劳的工作流程是什么？")
print(response)

四、Agent智能体

4.1 什么是Agent？

**Agent（智能体）**是能够自主规划、调用工具、执行任务的AI系统。

核心能力：

  • 规划：将复杂任务分解为子任务
  • 工具调用：调用API、数据库、代码执行器等
  • 记忆：保持对话上下文
  • 推理：根据情况进行决策

4.2 Agent应用场景

场景 说明
智能客服 自动回答问题，无法解决时转人工
数据分析 自动查询数据库，生成分析报告
代码助手 根据需求编写和调试代码
个人助理 管理日程、发送邮件、搜索信息

4.3 LlamaIndex中的Agent

from llama_index.core.agent import ReActAgent
from llama_index.llms.ollama import Ollama

# 加载模型
llm = Ollama(model="qwen3.5:0.8b")

# 创建Agent
agent = ReActAgent.from_tools(
    tools=[...],  # 工具列表
    llm=llm,
    verbose=True
)

# 执行任务
response = agent.chat("请帮我查询今天的天气")
print(response)

5.5 公司规章制度助手（09_公司规章制度助手.py）

from llama_index.llms.ollama import Ollama
from llama_index.embeddings.ollama import OllamaEmbedding
from llama_index.core import Settings, VectorStoreIndex
from llama_index.vector_stores.chroma import ChromaVectorStore
import chromadb

# 设置模型
Settings.llm = Ollama(model="qwen3.5:0.8b", base_url="http://localhost:11434")
Settings.embed_model = OllamaEmbedding(model_name="bge-m3:latest", base_url="http://localhost:11434")

# 连接向量库
db_client = chromadb.PersistentClient(path="vector_story/embedding_db")
db_collection = db_client.get_collection('test001')
vector_store = ChromaVectorStore(chroma_collection=db_collection)

# 创建索引
index = VectorStoreIndex.from_vector_store(vector_store)

# 创建查询引擎
query_engine = index.as_query_engine()

# 多轮对话
while True:
    question = input("请输入问题（输入'退出'结束）：")
    if question == "退出":
        break
    response = query_engine.query(question)
    print(f"回答：{response}")

六、关键知识点总结

6.1 大模型应用类型

类型 特点
Prompt Engineering 最基础，通过提示词引导
对话系统 增加记忆功能
RAG 外挂知识库，解决幻觉问题
Agent 具备推理和工具调用能力

6.2 RAG流程

文档加载 → 文本分块 → 文本向量化 → 索引构建 → 查询检索 → 生成答案

6.3 LlamaIndex核心组件

  • Data Connectors（数据加载）
  • Index（索引）
  • Retriever（检索）
  • Query Engine（查询引擎）
  • Agents（智能体）
  • Workflows（工作流）

6.4 向量数据库

ChromaDB：

  • 非关系型数据库
  • 专门用于存储和检索向量
  • 支持持久化存储

七、运行指南

7.1 环境准备

pip install llama-index llama-index-llms-ollama llama-index-embeddings-ollama chromadb

7.2 启动Ollama

确保Ollama服务已启动：

ollama serve

7.3 下载模型

ollama pull qwen3.5:0.8b
ollama pull bge-m3

7.4 运行示例

# 加载模型
python 01_通过llamaindex加载本地ollama中的模型.py

# 构建词向量库
python 06_词向量模型.py

# 进行检索
python 08_进行检索.py

# 公司规章制度助手
python 09_公司规章制度助手.py

总结

本课程系统讲解了：

  1. 大模型应用概念：Prompt Engineering、对话系统、RAG、Agent
  2. LlamaIndex框架：核心组件、安装使用、模型加载
  3. RAG系统搭建：文档加载、文本分块、向量化、索引构建、检索查询
  4. Agent智能体：规划、工具调用、记忆、推理

掌握LlamaIndex框架，是进行大模型应用开发的关键技能。
