Bash 命令汇总

查看 tail

tail -n+<N>     # 从第 N 行开始显示到末尾
tail -n <N>     # 显示末尾 N 行

查找 grep

grep 关键词 文件          # 文件中寻找关键词
grep -c 关键词 文件       # 只统计匹配到多少行
grep -n 关键词 文件       # 显示行号
grep -i 关键词 文件       # 忽略大小写
grep -v 关键词 文件       # 反向匹配
grep -rn 关键词 目录      # 递归搜索整个目录

查找分割 cut

cut -d分隔符 -f第几列 文件名

# 示例
cut -d: -f1 /etc/passwd      # 第一列
cut -d, -f1,3 data.csv       # 第 1、3 列
cut -d' ' -f1-5 log.txt      # 第 1 到 5 列

排序 sort

sort -n       # 按数值大小排序（小到大）
sort -nr      # 降序反转
sort -u       # 去重排列
sort -kX      # 指定排序第 X 列
  sort -k2,2  # 精确按照第 2 列排序
  sort -t',' -k2,2  # 指定分隔符

去重 uniq

[!warning]
只能删除掉相邻的重复内容，不能跨行去重（不挨着就没用了）

sort 文件名 | uniq -c    # 统计重复次数
sort 文件名 | uniq -u    # 输出唯一行，不重复的行

传参 xargs

基础用法

xargs -n1    # 执行完一个参数后执行下一个

示例：给定 file.txt

a.txt
b.log
c.jpg

执行 cat file.txt | xargs -n1 rm 等价于：

rm a.txt
rm b.log
rm c.jpg

占位替换符

cat list.txt | xargs -I {} cp {} /tmp/

等效于：

cp a.txt /tmp/
cp b.log /tmp/
cp c.jpg /tmp/
cat list.txt | xargs -I {} cp {} {}.bak

等效于：

cp a.txt a.txt.bak
cp b.log b.log.bak
