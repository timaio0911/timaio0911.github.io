Python 高级语法

Part1 – 深浅拷贝

[!info]

  • 直接拷贝：对象的引用，不产生拷贝——两个变量指向同一个地址
  • 浅拷贝：拷贝父对象，不会拷贝对象内部子对象。拷贝后只有第一层是独立
  • 深拷贝：完全拷贝了父对象及其子对象，拷贝后所有层是独立

拷贝规则

非容器类型无法拷贝

原子类型：int、float、str、bool、None 等，不存在拷贝概念，直接赋值就是引用。

元组变量

[!note]

  • 只包含原子类型：赋值、浅拷贝、深拷贝效果完全一样，都是引用复用
  • 包含嵌套容器类型：
    • 浅拷贝：元组本身新开辟空间，但内部嵌套容器还是引用同一份
    • 深拷贝：元组和内部所有嵌套容器都会递归新建，完全独立

列表/字典/集合（可变容器）

拷贝方式 说明
直接赋值 只是指向，共用同一份数据
浅拷贝 顶层容器新建，内层嵌套依旧引用
深拷贝 所有层级全部新建，完全隔离

浅拷贝 —— copy.copy()

import copy

a = [1, 2, [3, 4]]
b = copy.copy(a)

# 修改第一层元素：互不影响
b[0] = 99
print("a:", a)  # [1, 2, [3, 4]]
print("b:", b)  # [99, 2, [3, 4]]

# 修改内层嵌套列表：互相影响
b[2][0] = 88
print("改内层后 a:", a)  # [1, 2, [88, 4]]
print("改内层后 b:", b)  # [99, 2, [88, 4]]

[!tip]
常见浅拷贝方式：

  • copy.copy(对象)
  • 列表切片：b = a[:]
  • 字典：b = dict(a) / a.copy()
  • list()、dict() 构造器

深拷贝 —— copy.deepcopy()

import copy

a = [1, 2, [3, 4]]
b = copy.deepcopy(a)

# 修改内层嵌套：互不影响
b[2][0] = 88
print("a:", a)  # [1, 2, [3, 4]]
print("b:", b)  # [99, 2, [88, 4]]

[!warning]

  • 单层结构：浅拷贝够用
  • 嵌套结构：必须用深拷贝

对比示例

a = [1, 2, [3, 4]]

b1 = copy.copy(a)       # 浅拷贝
b2 = copy.deepcopy(a)   # 深拷贝

# 外层地址：全都不一样 → 都新建
print(id(a))
print(id(b1))
print(id(b2))

# 内层子列表
print(id(a[2]))
print(id(b1[2]))  # 浅拷贝：和原地址一样
print(id(b2[2]))  # 深拷贝：全新地址

Part2 – 迭代器

[!tip]
迭代器 = 书签

  1. 记住当前读到哪一页（记录遍历位置）
  2. 告诉你下一页是什么（提供下一个元素）
  3. 读完了告诉你没了（结束信号）

核心特点

  • 惰性计算：不用一次性把所有元素加载到内存，用一个取一个（省内存）
  • 记录位置：知道当前遍历到哪了，不会重复、不会乱序
  • 只能向前：只能往后遍历，不能回头、不能插队

可迭代对象 vs 迭代器

名称 解释 例子 能直接遍历？
可迭代对象 可以被迭代的"容器" 列表、字符串、元组、字典 能直接 for 循环
迭代器 用来遍历容器的"工具" 通过 iter() 得到的对象 能遍历，但只能遍历一次

[!note]

  • iter()：把可迭代对象 → 变成迭代器
  • next()：取出迭代器的下一个元素

基础用法

# 1. 可迭代对象（列表）
nums = [10, 20, 30]

# 2. 生成迭代器
it = iter(nums)

# 3. 用 next() 取值
print(next(it))  # 10
print(next(it))  # 20
print(next(it))  # 30

# 4. 取完了再调用 → 报错 StopIteration
# print(next(it))

for 循环本质

for x in [1, 2, 3]:
    print(x)

[!note]
for 循环 = 迭代器的语法糖

迭代器最大优势 —— 省内存

  • 列表 [1,2,3,...,100万]：一次性占 100 万空间
  • 迭代器：永远只存当前一个元素，用完就丢

处理超大文件、海量数据时，必须用迭代器！

自定义迭代器

规则：类里必须实现 __iter__() 和 __next__()

class MyIterator:
    def __init__(self, max_num):
        self.max = max_num
        self.current = 0

    def __iter__(self):
        return self

    def __next__(self):
        if self.current >= self.max:
            raise StopIteration
        num = self.current
        self.current += 1
        return num

# 使用
it = MyIterator(3)
print(next(it))  # 0
print(next(it))  # 1
print(next(it))  # 2

Part3 – 生成器

[!info]
生成器本质就是更简洁的迭代器，用 yield 就能快速造出一个迭代器。

yield vs return

特性 return yield
函数执行 直接结束 暂停，下次继续
返回值 返回后销毁 保留局部变量状态
内存 一次性返回 按需生成

基础用法

def my_generator(max_num):
    current = 0
    while current < max_num:
        yield current  # 暂停并返回值
        current += 1

g = my_generator(3)
print(next(g))  # 0
print(next(g))  # 1
print(next(g))  # 2

生成器表达式

# 列表推导式：一次性生成所有数据，占内存
lst = [x**2 for x in range(5)]

# 生成器表达式：惰性生成，用一个算一个
gen = (x**2 for x in range(5))

print(next(gen))  # 0
print(next(gen))  # 1

send() 传值

可以向生成器内部传值，是高级用法：

def gen():
    res = yield 1
    print("收到外部值：", res)
    yield 2

g = gen()
next(g)
g.send(666)  # 给 res 赋值为 666

Part4 – 装饰器

[!tip]
装饰器：在不修改原函数代码的前提下，给函数增加新功能。

基本语法

def 装饰器名(原函数):
    def 包装函数():
        # 这里写【新增的功能】
        原函数()  # 调用原来的函数
    return 包装函数

@装饰器名
def 原函数():
    pass

单层装饰器详解

第一步：理解本质

def outer(func):
    def x():
        ...
    return x

@outer
def send_wechat():
    print("微信")

[!note]
执行代码后 return x，此时相当于 send_wechat = outer(send_wechat)
之后只要执行 send_wechat() 都相当于执行 x()

第二步：添加功能

def outer(func):
    def x():
        print('程序开始')
        func()  # func() 就是原来的 send_wechat()
        print('程序结束')
    return x

@outer
def send_wechat():
    print("微信")

send_wechat()
# 输出：
# 程序开始
# 微信
# 程序结束

参数装饰器

def outer(func):
    def x(*args, **kwargs):  # 接住任何参数
        print('程序开始')
        func(*args, **kwargs)
        print('程序结束')
    return x

@outer
def send_wechat(body):
    print("微信", body)

send_wechat("你好")

返回值处理

def outer(func):
    def x(*args, **kwargs):
        print('程序开始')
        result = func(*args, **kwargs)  # 保存返回值
        print('程序结束')
        return result  # 返回结果
    return x

@outer
def send_wechat():
    print("微信")
    return 100

result = send_wechat()
print(result)  # 100

类装饰器

class Outer:
    def __init__(self, func):
        self.func = func

    def __call__(self):
        print('程序开始')
        self.func()
        print('程序结束')

@Outer
def send_wechat():
    print("微信")

send_wechat()

继承装饰器

class Outer:
    def __init__(self, func):
        self.func = func

    def __call__(self):
        print('程序开始')
        self.func()
        print('程序结束')

# 子类继承父类装饰器
class LogOuter(Outer):
    def __call__(self):
        print("【日志记录：准备执行】")
        super().__call__()
        print("【日志记录：执行完成】")

@LogOuter
def send_wechat():
    print("微信")

send_wechat()
# 输出：
# 【日志记录：准备执行】
# 程序开始
# 微信
# 程序结束
# 【日志记录：执行完成】
