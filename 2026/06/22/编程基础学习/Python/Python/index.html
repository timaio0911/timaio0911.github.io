Python 基础语法

Part1 – 输入与输出

解包赋值

A, B, C = map(int, input().split())

[!note]
默认按空格（包括多个空格、制表符、换行）分割字符串，把一整行字符串切成列表。

nums = list(map(int, input().split()))
# 输入：10 20 30 → [10,20,30]

# split()：按指定分隔符分割字符串

Part2 – 格式化输出

format 格式化

顺序占位

print("{} {}".format("Hello", "Python"))

下标占位

print("{1} {0}".format("A", "B"))  # B A

变量占位

print("{name} {age}".format(name="张三", age=18))

对齐与固定宽度

[!tip]
语法：{:宽度对齐方式}

  • < 左对齐
  • > 右对齐
  • ^ 居中
# 占12字符，左对齐
print("{:<12}".format("Province"))
# 占12字符，右对齐
print("{:>12}".format(139600.00))
# 居中
print("{:^10}".format("居中"))

小数保留

x = 3.14159
print("{:.2f}".format(x))  # 保留2位小数
print("{:.3f}".format(x))  # 保留3位小数

补零与补字符

# 总宽度5位，左边补0
print("{:05d}".format(12))   # 00012

# 用*填充居中
print("{:*^10}".format("OK"))  # ****OK****

进制转换

n = 25
print("二进制 {:b}".format(n))
print("八进制 {:o}".format(n))
print("十六进制 {:x}".format(n))

表格专用模板

print("{:<12}{:>12}{:>12}".format("Province", "Area(km2)", "Pop.(10K)"))
print("{:<12}{:>12.2f}{:>12.2f}".format("Anhui", 139600.0, 6461.0))

[!note]

  • 文字左对齐 :<12
  • 数字右对齐 + 保留 2 位小数 :>12.2f

f-string 简写

name = "李四"
age = 20
print(f"{name:<10} {age:>5}")

Part3 – 字符串与运算

常用字符串方法

s = "Hello Python"
print(s.lower())              # 转小写
print(s.upper())              # 转大写
print(s.replace("Hello", "Hi"))  # 替换
print("Python" in s)          # 判断子串是否存在
print(s.startswith("He"))     # 判断开头
print(s.endswith("on"))       # 判断结尾
print(s.split(" "))           # 按空格切分
print("-".join(["A", "B", "C"]))  # 连接列表为字符串

切片（slice）

[!info]
切片：从序列（list、tuple、字符串）中，截取一部分元素，生成一个新对象

序列[start : end : step]
  • start：起始下标（包含）
  • end：结束下标（不包含）
  • step：步长，隔几个取一个，默认 1

基本切片

规则：取左不取右

li = [10, 20, 30, 40, 50]
print(li[1:4])   # 下标1、2、3 → [20, 30, 40]

省略 start 或 end

print(li[:3])   # 从开头切到下标3 → [10, 20, 30]
print(li[2:])   # 从下标2切到末尾 → [30, 40, 50]
print(li[:])    # 整列表切片（浅拷贝）

步长与倒序

print(li[::2])    # 隔一个取一个 → [10, 30, 50]
print(li[::-1])   # 倒序 → [50, 40, 30, 20, 10]

负数下标切片

# 从倒数第3个 到 倒数第1个（不取）
print(li[-3:-1])  # [30, 40]

Part5 – 函数

语法结构

def 函数名(参数1, 参数2, ...):
    # 函数体（要执行的语句）
    return 返回值

不定长参数

[!info]

  • *args：不定长位置参数，用于接收任意数量的普通参数（打包成元组）
  • **kwargs：不定长关键字参数，用于接收任意数量的关键字参数（打包成字典）

*args 示例

def show_students(*names):
    print("学生名单如下：")
    for name in names:
        print("-", name)

show_students("张三", "李四", "王五")

**kwargs 示例

def show_info(**info):
    print("学生信息：")
    for key, value in info.items():
        print(f"{key}：{value}")

show_info(name="小明", age=18, city="北京")

混合使用

def func(a, b, *args, c=10, **kwargs):
    print("a =", a)
    print("b =", b)
    print("args =", args)
    print("c =", c)
    print("kwargs =", kwargs)

func(1, 2, 3, 4, c=20, x=100, y=200)

[!note]
在函数定义中，参数顺序必须为：**普通参数 → *args → 默认参数 → kwargs

闭包

[!tip]
闭包：内层函数记住了外层函数的变量，就算离开了外层函数，依然能继续用这个变量。

三个必备条件：

  1. 函数里嵌套函数（外层函数、内层函数）
  2. 内层函数引用了外层函数的局部变量
  3. 外层函数返回内层函数名（不调用、只返回）
def outer(x):
    # 外层变量 x
    def inner(y):
        # 内层用外层的 x
        return x + y
    # 返回内层函数
    return inner

# 把外层执行结果存到 f
f = outer(10)

# 现在 f 还能记住 x=10
print(f(5))   # 15
print(f(20))  # 30

[!note]

  • outer(10) 执行完本该销毁变量 x
  • 但因为 inner 用了 x，形成闭包
  • f 拿着 inner 永远记住了 x=10

匿名函数与高阶函数

匿名函数（lambda）

# 基本语法
lambda 参数列表 : 表达式

# 示例
square = lambda x: x ** 2
print(square(5))  # 25

map() 函数 —— 批量映射处理

nums = [1, 2, 3, 4, 5]
squares = list(map(lambda x: x ** 2, nums))

filter() 函数 —— 条件过滤

nums = [1, 2, 3, 4, 5, 6]
even_nums = list(filter(lambda x: x % 2 == 0, nums))
print("偶数列表：", even_nums)  # [2, 4, 6]

reduce() 函数 —— 累积计算

from functools import reduce

nums = [1, 2, 3, 4, 5]
product = reduce(lambda x, y: x * y, nums)
print("累乘结果：", product)  # 120

[!warning]
reduce 里的 lambda 永远要有两个参数

sorted() 函数 —— 自定义排序

words = ["python", "ai", "machine", "deep"]
result = sorted(words, key=lambda w: len(w))
print("按长度排序：", result)

Part6 – 类

面向对象编程（OOP）

[!info]
核心思想：以「事物/对象」为中心，把现实世界抽象成类和对象，用对象封装数据和行为。

# 类的定义
class Product:
    """商品类：描述商品的抽象概念"""
    
    # 类属性
    category = "电子产品"
    
    # 实例方法
    def __init__(self, name, price):
        self.name = name      # 实例属性
        self.price = price    # 实例属性
    
    def display(self):
        return f"{self.name}: ¥{self.price}"

构造方法 init

class Person:
    def __init__(self, name, age):
        self.name = name
        self.age = age

# 创建对象 → 自动调用 __init__
p1 = Person("小明", 20)
print(p1.name)  # 小明
print(p1.age)   # 20

类方法 @classmethod

class Student:
    total = 0  # 类属性，所有学生共享

    def __init__(self):
        Student.total += 1

    @classmethod
    def get_num(cls):
        print("总人数：", cls.total)

# 不用创建对象，直接类调用
Student.get_num()

静态方法 @staticmethod

class MathUtil:
    @staticmethod
    def add(a, b):
        return a + b

print(MathUtil.add(3, 5))

方法对比

类型 装饰器 首参数 归属
实例方法 无 self 实例对象
类方法 @classmethod cls 类
静态方法 @staticmethod 无 类

Part7 – 异常处理

常见内置异常

异常名称 触发原因
SyntaxError 语法错误
IndentationError 缩进错误
NameError 使用未定义的变量
TypeError 类型错误（数字 + 字符串）

异常处理语法

try:
    # 可能抛出异常的代码
    num = int(input("输入数字："))
    result = 100 / num
    print("结果：", result)

except ValueError:
    print("错误：你输入的不是整数！")

except ZeroDivisionError:
    print("错误：除数不能为 0！")

except Exception as e:
    print("未知错误：", e)

else:
    print("程序执行成功！")

finally:
    print("程序结束")

Part9 – 高级语法

深浅拷贝

[!info]

  • 直接拷贝：对象的引用，不产生拷贝
  • 浅拷贝：拷贝父对象，不会拷贝对象内部子对象
  • 深拷贝：完全拷贝了父对象及其子对象

浅拷贝示例

import copy

a = [1, 2, [3, 4]]
b = copy.copy(a)

# 修改第一层元素：互不影响
b[0] = 99
print("a:", a)  # [1, 2, [3, 4]]
print("b:", b)  # [99, 2, [3, 4]]

# 修改内层嵌套列表：互相影响
b[2][0] = 88
print("改内层后 a:", a)  # [1, 2, [88, 4]]
print("改内层后 b:", b)  # [99, 2, [88, 4]]

深拷贝示例

import copy

a = [1, 2, [3, 4]]
b = copy.deepcopy(a)

# 修改内层嵌套：互不影响
b[2][0] = 88
print("a:", a)  # [1, 2, [3, 4]]
print("b:", b)  # [99, 2, [88, 4]]

迭代器

[!tip]
迭代器 = 书签

  1. 记住当前读到哪一页
  2. 告诉你下一页是什么
  3. 读完了告诉你没了

可迭代对象 vs 迭代器

名称 解释 例子 能直接遍历？
可迭代对象 可以被迭代的"容器" 列表、字符串、元组、字典 能直接 for 循环
迭代器 用来遍历容器的"工具" 通过 iter() 得到的对象 能遍历，但只能遍历一次

基础用法

nums = [10, 20, 30]
it = iter(nums)

print(next(it))  # 10
print(next(it))  # 20
print(next(it))  # 30

自定义迭代器

class MyIterator:
    def __init__(self, max_num):
        self.max = max_num
        self.current = 0

    def __iter__(self):
        return self

    def __next__(self):
        if self.current >= self.max:
            raise StopIteration
        num = self.current
        self.current += 1
        return num

it = MyIterator(3)
print(next(it))  # 0
print(next(it))  # 1
print(next(it))  # 2

生成器

[!note]
生成器本质就是更简洁的迭代器，用 yield 就能快速造出一个迭代器。

yield vs return

  • return：函数直接结束，返回值，销毁局部变量
  • yield：暂停函数，返回值；下次迭代从暂停处继续执行
def my_generator(max_num):
    current = 0
    while current < max_num:
        yield current
        current += 1

g = my_generator(3)
print(next(g))  # 0
print(next(g))  # 1
print(next(g))  # 2

生成器表达式

# 列表推导式：一次性生成所有数据，占内存
lst = [x**2 for x in range(5)]

# 生成器表达式：惰性生成，用一个算一个
gen = (x**2 for x in range(5))

装饰器

[!tip]
装饰器：在不修改原函数代码的前提下，给函数增加新功能。

基本语法

def 装饰器名(原函数):
    def 包装函数():
        # 新增的功能
        原函数()  # 调用原来的函数
    return 包装函数

@装饰器名
def 原函数():
    pass

示例

def outer(func):
    def x(*args, **kwargs):
        print('程序开始')
        result = func(*args, **kwargs)
        print('程序结束')
        return result
    return x

@outer
def send_wechat(body):
    print("微信", body)
    return "发送成功"

result = send_wechat("你好")
print(result)

类装饰器

class Outer:
    def __init__(self, func):
        self.func = func

    def __call__(self):
        print('程序开始')
        self.func()
        print('程序结束')

@Outer
def send_wechat():
    print("微信")

send_wechat()
