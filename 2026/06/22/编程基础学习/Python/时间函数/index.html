Python 时间操作

Part1 – time 模块

常用函数

函数 作用 示例
time.time() 获取当前时间戳（1970年至今的秒数） timestamp = time.time()
time.sleep(秒数) 程序暂停指定秒数 time.sleep(2)
time.ctime([时间戳]) 时间戳转易读字符串 time.ctime() → Wed Oct 11 15:30:00 2025
time.localtime([时间戳]) 时间戳转本地时间结构体 local_time = time.localtime()
time.strftime(格式, 结构体) 自定义格式化时间（最常用） 见下方示例
time.strptime(字符串, 格式) 字符串解析为时间结构体 见下方示例

格式化时间

from datetime import datetime

now = datetime.now()
# 格式：年-月-日 时:分:秒
res = now.strftime("%Y-%m-%d %H:%M:%S")
print(res)

解析时间字符串

# 把字符串解析为 datetime 对象
dt = datetime.strptime("2025-10-11 16:00:00", "%Y-%m-%d %H:%M:%S")

[!tip]
常用格式化符号：

  • %Y：4位年（2025）
  • %m：2位月（10）
  • %d：2位日（11）
  • %H：24小时制（16）
  • %M：分钟（30）
  • %S：秒（00）
  • 常用组合：%Y-%m-%d %H:%M:%S

Part2 – datetime 模块（最常用）

获取当前时间

from datetime import datetime, date

# 获取当前日期 + 时间
now = datetime.now()

# 仅获取当前日期（无时分秒）
today = date.today()

时间加减计算

[!info]
timedelta 支持天/小时/分钟/秒级别的计算，不支持直接加减月/年

from datetime import datetime, timedelta

now = datetime.now()

# 加 3 天
later = now + timedelta(days=3)

# 减 2 小时
before = now - timedelta(hours=2)

# 组合：1天2小时30分钟
mix = now + timedelta(days=1, hours=2, minutes=30)

时间对象属性

dt = datetime.now()

print(dt.year)     # 年
print(dt.month)    # 月
print(dt.day)      # 日
print(dt.hour)     # 时
print(dt.minute)   # 分
print(dt.second)   # 秒

时间戳互转

# 时间对象 → 时间戳
dt = datetime.now()
timestamp = dt.timestamp()

# 时间戳 → 时间对象
dt = datetime.fromtimestamp(1728627000)

Part3 – calendar 模块

常用函数

函数 作用
calendar.isleap(年份) 判断是否为闰年
calendar.weekday(年, 月, 日) 返回星期（0=周一，6=周日）
calendar.month(年, 月) 打印月日历

示例

import calendar

# 判断闰年
print(calendar.isleap(2024))  # True

# 获取星期
print(calendar.weekday(2025, 10, 11))  # 5 = 周六

# 打印月日历
print(calendar.month(2025, 10))

常用时间操作汇总

[!note]
获取当前时间：

from datetime import datetime
now = datetime.now()

[!tip]
格式化输出：

print(now.strftime("%Y-%m-%d %H:%M:%S"))

[!info]
时间计算：

from datetime import timedelta
tomorrow = now + timedelta(days=1)
