异常处理与模块化

Part1 – 错误和异常

语法错误（SyntaxError）

[!warning]

  • 原因：代码写法不符合 Python 语法规则（少冒号、缩进错、引号不匹配等）
  • 时机：程序运行前，解释器检查代码时就报错
  • 特点：必须修改代码才能运行，无法用程序捕获处理

异常（Exception）

[!info]

  • 原因：代码语法正确，但运行时出现问题（除零、索引越界、文件不存在等）
  • 时机：程序运行过程中触发
  • 特点：可以用 try/except 捕获处理，让程序不崩溃

Python 常见内置异常

异常名称 触发原因
SyntaxError 语法错误
IndentationError 缩进错误
NameError 使用未定义的变量
TypeError 类型错误（数字 + 字符串）
ValueError 值类型正确，但值无效
IndexError 列表/元组索引超出范围
KeyError 字典键不存在
ZeroDivisionError 除数为0
FileNotFoundError 文件不存在
AttributeError 对象没有该属性/方法

异常层次结构

BaseException
├── SystemExit          # sys.exit()调用
├── KeyboardInterrupt   # Ctrl+C中断
└── Exception           # 所有常规异常的基类
    ├── StopIteration   # 迭代器耗尽
    ├── ArithmeticError # 算术错误
    │   ├── FloatingPointError
    │   ├── OverflowError
    │   └── ZeroDivisionError
    ├── LookupError     # 索引/键错误
    │   ├── IndexError
    │   └── KeyError
    ├── OSError         # 操作系统错误
    │   ├── FileNotFoundError
    │   ├── PermissionError
    │   └── TimeoutError
    ├── ValueError      # 值错误
    ├── TypeError       # 类型错误
    ├── AttributeError  # 属性错误
    ├── NameError       # 名称未定义
    ├── SyntaxError     # 语法错误
    ├── IndentationError # 缩进错误
    └── RuntimeError    # 运行时错误

异常处理语法

基本格式

try:
    # 可能抛出异常的代码
    风险代码块
except 异常类型:
    # 捕获到对应异常后执行的代码
    处理异常代码

完整格式

try:
    # 可能出错的代码
    num = int(input("输入数字："))
    result = 100 / num
    print("结果：", result)

except ValueError:
    print("错误：你输入的不是整数！")

except ZeroDivisionError:
    print("错误：除数不能为 0！")

except Exception as e:
    print("未知错误：", e)

else:
    print("程序执行成功！")

finally:
    print("程序结束")

[!note]

  • try：包含可能出错的代码
  • except：捕获指定异常
  • else：没有异常时执行（可选）
  • finally：无论是否异常都会执行（常用于关闭文件、释放资源）

assert 断言

assert 条件表达式, 错误提示信息

[!warning]

  • 条件为 True：啥也不做，程序继续
  • 条件为 False：直接抛出 AssertionError 异常
def calc_age(age):
    assert age > 0, "年龄不能小于等于0"
    print("年龄合法，继续计算")

calc_age(-5)  # AssertionError: 年龄不能小于等于0

[!tip]

  • 专门用于调试、内部校验，不用于处理业务异常
  • 可以加 -O 参数关闭断言（生产环境会失效）
  • 命令行：python -O 你的脚本.py

主动抛出异常 raise

def check_age(age):
    if age < 0:
        raise ValueError("年龄不能为负数！")
    print("年龄合法")

try:
    check_age(-5)
except ValueError as e:
    print(e)  # 年龄不能为负数！

多异常捕获

try:
    a = int(input("请输入整数 a: "))
    b = int(input("请输入整数 b: "))
    result = a / b
    print(f"计算结果: {result}")

except (ValueError, ZeroDivisionError) as e:
    err_name = type(e).__name__
    print(f"发生错误：{err_name}")

finally:
    print("程序结束")

自定义异常

当内置异常不够用时，可以创建自己的异常类，继承 Exception 即可

class AgeError(Exception):
    def __init__(self, msg):
        self.message = msg
        super().__init__(self.message)

def check(age):
    if age < 0:
        raise AgeError("年龄异常：不能小于0")

try:
    check(-10)
except AgeError as e:
    print(e)  # 年龄异常：不能小于0

with 语句

[!tip]
不管里面代码正常执行还是报错崩溃，一定自动关闭文件，不用你写 close()

with open("test.txt", "r", encoding="utf-8") as f:
    content = f.read()
# 出了这一行，自动帮你 close()

Part2 – 模块化

模块与包

[!info]

  • 模块（module）：单个 .py 文件，就是一个模块
  • 包（package）：放一堆模块的文件夹，里面要有 __init__.py

Python 模块搜索顺序

  1. 先找当前目录（最优先）
  2. 再找环境变量里的目录（PYTHONPATH）
  3. 再找 Python 安装目录（内置模块：sys、os、random）
  4. 最后找第三方库目录（pip install 安装的模块）

导入方式

创建模块 calc.py

def add(a, b):
    return a + b

def sub(a, b):
    return a - b

导入使用

# 导入整个模块
import calc
print(calc.add(3, 5))

# 只导入指定功能
from calc import add
print(add(2, 6))

# 导入改名（别名）
import calc as c
print(c.add(1, 2))

all 关键字

[!note]
控制别人用 from 模块 import * 时，到底能导入哪些东西

# test.py
__all__ = ["add", "name"]  # 只允许 import * 导入这两个

name = "小明"
age = 20

def add():
    print("加法")

def sub():
    print("减法")
# 另一个文件
from test import *

print(name)  # 可以用
add()        # 可以用

print(age)   # 报错！找不到
sub()        # 报错！找不到

name 关键字

[!tip]
__name__ 是一个内置变量：

  • 文件自己直接运行时：__name__ == "__main__"
  • 文件被别人导入当模块时：__name__ == 模块名
# 写在每个py文件最后
if __name__ == "__main__":
    # 这里的代码：只有自己直接运行才执行
    # 被别人import导入时，这一段完全不执行
    print("我只在本文件直接运行时执行")

dir() 关键字

# 查看当前作用域里有哪些名字
print(dir())

# 查看对象有哪些可用的属性和方法
print(dir(str))

包的使用

我的包/
    __init__.py   ← 标志这是一个Python包
    工具.py
    时间.py
    计算.py

导入方式

# 导入整个包
import 包名

# 从包里导入某个文件（模块）
from 包名 import 模块名

# 从包里直接导入函数
from 包名.模块名 import 函数名
