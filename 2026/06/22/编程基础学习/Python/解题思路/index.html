最小堆

堆本质

完全二叉树，底层用列表 / 数组存储，Python 只有最小堆，无原生最大堆

最小堆（小顶堆）

根节点值最小，任意父节点 ≤ 子节点，堆顶 = 全局最小值

最大堆（大顶堆）

根节点值最大，任意父节点 ≥ 子节点，堆顶 = 全局最大值

Python heapq 核心用法

import heapq
heap = []
heapq.heappush(heap, x)  # 元素入堆，自动维护堆结构
						# heap -> heap[]提供一个堆容器
    					# x -> 要录入的元素（元组，整形，字符串...）
heapq.heappop(heap)     # 弹出堆顶最小元素，自动重构堆
heapq.heapify(list)     # 列表原地转为小堆，O(n)
heapq.heappushpop()     # 先压入再弹出堆顶

堆存取规则

  1. 堆内元素必须支持大小比较

  2. 自定义对象（链表节点）不能直接比较

    固定封装格式：

    (排序键, 唯一下标, 对象)

    例：

    (node.val, idx, node)

    避免同值报错
