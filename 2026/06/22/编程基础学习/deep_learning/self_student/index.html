深度学习基础

三步走

Part1 – Function With Unknown Parameters

[!info]
寻找一个含有未知参数的方程式

  • model：带有未知参数的公式
  • feature：特征，一般充当 x

未知参数（Unknown Parameters）

  • weight：权重（w）—— 要寻找的
  • bias：偏置（b）—— 要寻找的
# 最简单的神经元
输出 = 权重 × 输入 + 偏置
y = w × x + b

Part2 – Define Loss From Training Data

[!info]
损失函数的输入值是 w, b → L(b, w)
损失函数的输出值反映这一组 w, b 是好还是不好

预测值 y 与真实值 ŷ（label）作比较，其差称之为 e

平均绝对值误差（MAE）

$$
\text{MAE} = \frac{1}{n}\sum_{i=1}^n \left| y_i - \hat{y}_i \right|
$$

均方误差（MSE）

$$
\text{MSE} = \frac{1}{n}\sum_{i=1}^n \big( y_i - \hat{y}_i \big)^2
$$

Part3 – Optimization

[!tip]
使用 Gradient Descent（梯度下降） 优化寻找最好的 w, b

  • 通过对斜率的计算，控制参数向斜率更小的地方移动
  • 移动的每步步长受到斜率和学习效率（η）的影响
  • 超参数（Hyperparameters）：人为预先设定，训练全程不变

依次计算更新 w 的值，加入 b 后的计算方法是一样的，微分后减去

进阶

分段曲线拟合

[!note]
分段曲线 = constant = sum of a set of linear curves

类似于用多段直线可以拟合出一个曲线，多个 sigmoid 相组合可以得到复杂曲线

Sigmoid 函数：

$$
\sigma(z) = \frac{1}{1+e^{-z}}
$$

当引入更多的数据（比如用前三天的信息一起来预测）：

  • j：表示第几个特征值（第几天的数值）
  • i：表示一个蓝色的方程（用来组合）

整合后向量计算，又变成了最初的格式

单维未知参数

[!info]
网络中所有权重 W、偏置 b 统称为未知参数

把每层的矩阵/向量参数，按顺序展平、拼接成单个一维长向量：
θ = [w1, b1, w2, b2, …, wL, bL]ᵀ

Optimization of New Model

每个数值分别对 L 求偏导，算法和原来一样

在实际计算过程中会先对数据进行分组：

  • 分成 n 份 batch
  • batch 也是一个超参数

验证

交叉验证（Cross Validation）

[!tip]
将数据集随机分成 K 个大小相近的子集（折），每次选择其中 1 折作为验证集，其余 K-1 折作为训练集

数据集：[1,2,3,4,5] 五个子集

第1次：训练[2,3,4,5] → 验证[1] → 准确率 0.85
第2次：训练[1,3,4,5] → 验证[2] → 准确率 0.87
第3次：训练[1,2,4,5] → 验证[3] → 准确率 0.83
第4次：训练[1,2,3,5] → 验证[4] → 准确率 0.86
第5次：训练[1,2,3,4] → 验证[5] → 准确率 0.84

最终分数 = (0.85+0.87+0.83+0.86+0.84)/5 = 0.85

Optimization 相关知识

二阶泰勒展开近似

损失函数 L(θ) 在参数点 θ=θ′ 处的二阶泰勒展开：

$$
L(\theta) \approx L(\theta’) + (\theta - \theta’)^T g + \frac{1}{2} (\theta - \theta’)^T H (\theta - \theta’)
$$

各项含义：

  • $L(\theta’)$：参数在 θ′ 处的损失值（基准点）
  • 一阶项 $(\theta - \theta’)^T g$
    • $g = \nabla L(\theta’)$ 是梯度向量
    • 描述函数在 θ′ 处的局部斜率
  • 二阶项 $\frac{1}{2} (\theta - \theta’)^T H (\theta - \theta’)$
    • H 是 Hessian 矩阵
    • 描述函数在 θ′ 处的曲率/凹凸性

Hessian 矩阵

[!info]
Hessian 矩阵是一个多元函数的二阶偏导数构成的方阵，用于描述损失函数曲面的曲率

直观理解：

  • 一阶导数（梯度）：告诉你函数在哪个方向是下坡（速度）
  • 二阶导数（Hessian）：告诉你下坡的坡度是变陡还是变缓（加速度）

类比：你站在山坡上

  • 梯度：脚下的斜坡有多陡，往哪边滑
  • Hessian：往前一步，斜坡是变得更陡还是更平

如何理解 Hessian

当走到驻点时，g 的值为零（一阶导函数为 0），此时根据二阶导数判断所在位置的附近形状

vᵀHv 的符号 损失变化 函数形态 驻点类型
始终为正 L(θ) > L(θ′) 所有方向都是凸的（碗状） 局部极小值
始终为负 L(θ) < L(θ′) 所有方向都是凹的（山状） 局部极大值
有正有负 有的方向升、有的方向降 有的方向凸、有的方向凹 鞍点

[!tip]
沿着 Hessian 矩阵的负特征值对应的特征向量方向移动，损失函数会下降
